This is an open sourced visual tutorial on how to:

  • register and setup a new domain using OnlyDomains
  • host unlimited number of subdomains using Cloudflare DNS
  • configure 11ty using Industrial Empathy template
  • deploy serverless via GitHub Actions
  • host content as code using GitHub Pages
  • deliver at edge using CloudFlare CDN
  • edit markdown content using Netlify CMS

This blog template is based on eleventy-high-performance-blog-plus an example repository implementation of eleventy-high-performance-blog which is based on eleventy-base-blog.

You can find all the terribly messy commits for this demo on github and all the images used throughout this blog on this gist. You can also find some of my other work over at github:0-vortex.
